## Youtube Media Clone
made with React under node v16.13.2

### Rapid API:
[Youtube v3](https://rapidapi.com/ytdlfree/api/youtube-v31/) api endpoints

### Helper library:
[MUI React Kit](https://mui.com/) documentation

### Dependencies:
- Material UI
    - @emotion/react
    - @emotion/styles
- MUI Kit (latest v5)
    - @mui/icons-material
    - @mui/material
- API calls
    - axios
- Routing (latest v6)
    - react-router-dom

### Deployment:
[Netlify](https://my-ytmedia.netlify.app/) project
